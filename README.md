
**苏格拉底实验室**（The Socrates Lab）是南京大学新闻新闻传播学院王成军老师所领导的一个虚拟研究团队，主要成员包括王成军、他的学生以及其他合作者。

2. 王成军维护了[“计算传播学豆瓣小组”](https://www.douban.com/group/webmining/)，每天分享其个人的研究观察和想法。
